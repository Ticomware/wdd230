Typography

This website uses two font families. Almost all content on this website uses the Segoe UI font from Microsoft. It is a very pretty sans-serif font available on all modern versions Microsoft Windows. If the Segoe UI font is not available, it will use the San Francisco font (used on macOS and iOS.) After that it will fall back to Helvetica Neue, Roboto (Android default), Oxygen, Ubuntu, Cantarell, Open Sans, and finally the default sans-serif font on the system.

The default font for this website.

For the name of the organization in the header, I decided to use Baskerville. It’s a font from the mid-1700s that makes the chamber appear more trustworthy. If Baskerville isn’t available, it will fall back to Hoefler Text, then Garamond, then Times New Roman, and finally the default serif font.

The quick brown fox jumped over the lazy dog.

The font used for the organization name in the header.

Unless it is required for this project, I will avoid using Google fonts. Google is extremely untrustworthy with user data, and loading external fonts increases the memory and bandwidth impact of the website on the client. I believe in making websites as accessible as possible. So much of the modern web is less accessible than it was fifteen years ago because web developers focused on making websites trendier and ignored customers with older hardware. In my own browser, I disable any unnecessary JavaScript.
